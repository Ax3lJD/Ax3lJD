What drew me to computer science wasn't any one thing. It was the range. The same discipline lets you dig into low-level systems, build something a customer uses tomorrow, or rethink how an entire team works. I like all of it. I like the problem solving, I like that the problems are always different, and I like that the best solutions usually come from understanding the people involved, not just the code.

I didn't take the most direct path here. I co-owned an ice cream shop in New Orleans, maintained web apps for a mass spectrometry lab at Tulane, and spent a summer writing Python automation against COBOL systems at Texas Farm Bureau. Each one gave me a different lens on how technology meets real people and real operations. That range is what I want my career to look like too.

## What I've been building

**[Yami Hibachi](https://yamihibachi.com)** · Vue 3 + Spring Boot + Square Payments. Full online ordering platform for a local restaurant. Promo codes, scheduled pickup, email receipts. Live and taking real orders.

**Cocoon** · Senior capstone for Microsoft Consulting Services. AI-powered SOW review using Neo4j graph RAG and ML risk classification. I handled infrastructure, QA, and Azure deployment.

**[DNS Server](https://github.com/zhansu)** · Built from scratch in Rust per RFC 1035, no high-level DNS libraries. Dual-stack IPv4/IPv6, recursive resolution, thread-safe caching. Probably my favorite project. There's something really satisfying about implementing a protocol you use every day without thinking about it.

**TCP Implementation** · Reliable transport protocol in Rust. Sliding window, retransmission, out-of-order packet handling.

**PlastiScan** · Mobile app (Expo React Native + Express.js) using Claude's vision API to estimate microplastic exposure from food photos. Work in progress.

## Work

**Software Developer Intern @ [Texas Farm Bureau](https://www.txfb-ins.com/)** · Wrote Python automation for COBOL enterprise systems serving 500K+ member accounts. Sat with the people using the systems every day, figured out what was slowing them down, and built tools that actually fixed it.

**Web Developer Intern @ Tulane University** · Maintained web apps for the Mass Spectrometry Lab. Gave bi-weekly presentations to supervisors, which did more for my communication skills than any class.

**Co-Owner @ OK Ice Cream, New Orleans** · Hired and trained staff, managed daily operations, grew the customer base 20% and sales 500%. Taught me that the best work happens when you listen first and take ownership of problems, not just tasks.
