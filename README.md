I didn't take the most direct path to software engineering. I co-owned an ice cream shop in New Orleans, maintained web apps for a mass spectrometry lab at Tulane, and spent a summer writing Python automation against COBOL systems at Texas Farm Bureau. All very different, but each one taught me something about building things that actually help people and working alongside folks who think differently than I do.

The thing that gets me excited is when something I build saves someone real time or makes their day a little smoother. That's the through line, whether it's a script or a full product.

## What I've been building

**[Yami Hibachi](https://yamihibachi.com)** · Vue 3 + Spring Boot + Square Payments. Full online ordering platform for a local restaurant. Promo codes, scheduled pickup, email receipts. Live and taking real orders.

**Cocoon** · Senior capstone for Microsoft Consulting Services. AI-powered SOW review using Neo4j graph RAG and ML risk classification. I handled infrastructure, QA, and Azure deployment.

**[DNS Server](https://github.com/zhansu)** · Built from scratch in Rust per RFC 1035, no high-level DNS libraries. Dual-stack IPv4/IPv6, recursive resolution, thread-safe caching. Probably my favorite project. There's something really satisfying about implementing a protocol you use every day without thinking about it.

**TCP Implementation** · Reliable transport protocol in Rust. Sliding window, retransmission, out-of-order packet handling.

**PlastiScan** · Mobile app (Expo React Native + Express.js) using Claude's vision API to estimate microplastic exposure from food photos. Work in progress.

## Work

**Software Developer Intern @ [Texas Farm Bureau](https://www.txfb-ins.com/)** · Python automation for COBOL enterprise systems serving 500K+ member accounts. Got to see firsthand how one well-placed script can save a whole team hours of manual work every week.

**Web Developer Intern @ Tulane University** · Maintained web apps for the Mass Spectrometry Lab. Gave bi-weekly presentations to supervisors, which did more for my communication skills than any class.

**Co-Owner @ OK Ice Cream, New Orleans** · Hired and trained staff, managed daily operations, grew the customer base 20% and sales 500%. Running a small business taught me that leadership is less about having all the answers and more about showing up when things don't go as planned.
